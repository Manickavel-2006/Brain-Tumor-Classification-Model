# 🧠 Brain Tumor Classification Using Deep Learning & Grad-CAM

This project focuses on **brain tumor classification from MRI images** using **Deep Learning** techniques and **Grad-CAM (Gradient-weighted Class Activation Mapping)** for model interpretability.

The notebook demonstrates how a trained deep learning model can not only classify brain tumors but also **visually explain its predictions**, which is crucial for medical and clinical applications.

---

## 🚀 Project Objectives

- Classify brain MRI images into tumor-related categories
- Use Deep Learning models for accurate image classification
- Apply **Grad-CAM** to highlight important regions influencing predictions
- Improve model transparency and trust in medical AI systems

---

## 🧠 Key Features

- MRI image preprocessing
- Deep Learning–based classification
- Model prediction visualization
- **Grad-CAM heatmaps** for explainable AI
- End-to-end implementation in a single Jupyter Notebook

---

## 🛠️ Technologies & Libraries Used

- Python
- TensorFlow / Keras
- NumPy
- OpenCV
- Matplotlib
- Grad-CAM
- Jupyter Notebook
